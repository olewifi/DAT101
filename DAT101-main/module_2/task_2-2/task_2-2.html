1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. When you know you want the input to be constant, and not change you should put in a constant (const). When you know the input is gonna be changed you should use variables (let). If you are not going to change the input they all can be constants. But mostly one does not want to hard code the code, but make it flexible and have the possibility to make changes easily. Therefore one usually don't use const for every input.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. We have different methods to choose the given number of decimals. One can use for example toFixed(), toPrecision() and Math.round(). The first two will give the exact decimals that are written. Math.round will round up or down the number.

    5. We can use comparison operators to compare the numbers. We can check them if they are equal by using "==". 12.199999 is not the same as 12.2 unless you round up the first number.
